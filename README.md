# Bing Homepage Quiz 数据储存库

本项目用于每日自动获取并储存题目数据

> Action 会在北京时间零点一分和二十三点各运行一次，前者是为了及时获取数据，后者是为了确保题目正确率被正确统计

## Web API

#### 获取历史题目数据
`/history/{{YY-MM-DD}}.json`

#### 获取今日题目数据
`/current.json`
