# Bing Homepage Quiz 数据储存库

本项目用于每日自动获取并储存题目数据

## Web API

#### 获取历史题目数据
`/history/{{YY-MM-DD}}.json`

#### 获取今日题目数据
`/current.json`

    "date": "2025-08-19", // 该题目的日期
